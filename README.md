# 🔢 Multiplication Table Generator

## 📖 Description
The **Multiplication Table Generator** is a Java console application that prompts the user to enter a number and then displays its multiplication table from **1 to 10** using a **for loop**.

---

## ✨ Features
- 🔢 Accepts user input using the `Scanner` class
- 🔁 Uses a `for` loop to generate a multiplication table
- ➗ Performs multiplication calculations
- 📊 Displays results in an easy-to-read table format

---

## 🛠️ Technologies Used
- ☕ Java
- 💻 Eclipse IDE

---

## ▶️ How to Run
1. Run the program.
2. Enter a number when prompted.
3. View the multiplication table from **1 to 10**.
